EM算法

概率是特定环境下某件事情发生的可能性，即结果没产生之前依据环境所对应的参数来预测某件事情发生的可能性

似然是在确定的结果下去推测产生这个结果的可能环境(参数)

  • 三硬币模型可以写作：

    P(y∣θ)=∑zP(y,z∣θ)=∑zP(z∣θ)P(y∣z,θ)=πpy(1−p)1−y+(1−π)qy(1−q)1−yP(y|\theta)=\sum_z P(y,z|\theta)=\sum_z P(z|\theta)P(y|z,\theta)=\pi p^y(1-p)^{1-y}+(1-\pi)q^y(1-q)^{1-y}P(y∣θ)=∑z​P(y,z∣θ)=∑z​P(z∣θ)P(y∣z,θ)=πpy(1−p)1−y+(1−π)qy(1−q)1−y

    对应的似然函数P(Y∣θ)=∑ZP(Z∣θ)P(Y∣Z,θ)P(Y|\theta)=\sum_Z P(Z|\theta)P(Y|Z,\theta)P(Y∣θ)=∑Z​P(Z∣θ)P(Y∣Z,θ)

    即P(Y∣θ)=∏j=1n[πpyj(1−p)1−yj+(1−π)qyj(1−q)1−yj]P(Y|\theta)=\prod_{j=1}^n[\pi p^{y_j}(1-p)^{1-{y_j}}+(1-\pi)q^{y_j}(1-q)^{1-{y_j}}]P(Y∣θ)=∏j=1n​[πpyj​(1−p)1−yj​+(1−π)qyj​(1−q)1−yj​]

问题没有解析解，只能迭代求解。EM算法就是用于求解这个问题的迭代算法。

EM算法：

  1. 选取初始值：θ(0)=(π(0),p(0),q(0))\theta^{(0)}=(\pi^{(0)},p^{(0)},q^{(0)})θ(0)=(π(0),p(0),q(0))

  2. E步：计算Q函数。θ(i)\theta^{(i)}θ(i)为第i次迭代参数θ\thetaθ的估计值，在第i+1次迭代的E步，计算：

    Q(\theta,\theta^{(i)})=E_Z[\log P(Y,Z|\theta)|Y,\theta^{(i)}]\\ =\underset Z\sum\log P(Y,Z|\theta^{(i)})P(Z|Y,\theta^{(i)})

  3. M步：求使Q(θ,θ(i))Q(\theta,\theta^{(i)})Q(θ,θ(i))极大化的θ\thetaθ，确定第i+1次迭代的参数的估计值θ(i+1)\theta^{(i+1)}θ(i+1)

    \theta^{(i+1)}=\arg \underset{\theta}\max Q(\theta,\theta^{(i)})

  4. 重复第2 3步直到收敛

    停止迭代的条件一般是对于较小的正数ϵ1,ϵ2\epsilon_1,\epsilon_2ϵ1​,ϵ2​，若满足

    ∣∣θ(i+1)−theta(i)∣∣<ϵ1或∣∣Q(θ(i+1),θ(i))−Q(θ(i),θ(i))∣∣<ϵ2||\theta^{(i+1)}-theta^{(i)}||<\epsilon_1或||Q(\theta^{(i+1)},\theta^{(i)})-Q(\theta^{(i)},\theta^{(i)})||<\epsilon_2 ∣∣θ(i+1)−theta(i)∣∣<ϵ1​或∣∣Q(θ(i+1),θ(i))−Q(θ(i),θ(i))∣∣<ϵ2​

    则停止迭代

import numpy as np
import math

class EM:
    def __init__(self, theta, y):
        self.pi, self.p, self.q = theta
        self.y = y  # 初始化观测数据

    # E步
    def mu(self, j):
        prob_1 = self.pi * (self.p ** self.y[j]) * ((1 - self.p) ** (1 - self.y[j]))
        prob_2 = (1 - self.pi) * (self.q ** self.y[j]) * ((1 - self.q) ** (1 - self.y[j]))
        return prob_1 / (prob_1 + prob_2)

    # M步
    def M_step(self, K):
        n = len(self.y)
        for k in range(K):
            mu_vals = [self.mu(j) for j in range(n)]

            self.pi = sum(mu_vals) / n
            self.p = sum(mu_vals[j] * self.y[j] for j in range(n)) / sum(mu_vals)
            self.q = sum((1 - mu_vals[j]) * self.y[j] for j in range(n)) / sum(1 - mu for mu in mu_vals)
            
            print(f"第{k+1}次迭代: pi={self.pi:.4f}, p={self.p:.4f}, q={self.q:.4f}")
        return self.pi, self.p, self.q

# 数据：掷硬币的观测结果
y = [1, 1, 0, 1, 0, 0, 1, 0, 1, 1]

# 两组不同初始参数

em1.M_step(K=10)

import numpy as np
import time

def generate_data(mu0, sigma0, mu1, sigma1, alpha0, alpha1):
    '''
    初始化数据集，通过高斯分布生成两个混合数据集。
    '''
    assert abs(alpha0 + alpha1 - 1.0) < 1e-6, "alpha0 和 alpha1 之和必须为 1"
    length = 1000
    data0 = np.random.normal(mu0, sigma0, int(length * alpha0))
    data1 = np.random.normal(mu1, sigma1, int(length * alpha1))
    data = np.concatenate((data0, data1))
    np.random.shuffle(data)  # 使用 numpy 的打乱函数
    return data

def gaussian_prob(x, mu, sigma):
    '''
    高斯概率密度函数
    '''
    coeff = 1.0 / (np.sqrt(2 * np.pi) * sigma)
    exponent = np.exp(-((x - mu) ** 2) / (2 * sigma ** 2))
    return coeff * exponent

def E_step(data, alpha0, mu0, sigma0, alpha1, mu1, sigma1):
    '''
    E步：计算每个数据点属于两个高斯分布的概率（响应度）
    '''
    p0 = alpha0 * gaussian_prob(data, mu0, sigma0)
    p1 = alpha1 * gaussian_prob(data, mu1, sigma1)
    total = p0 + p1
    # 避免除以零
    total[total == 0] = 1e-10
    gamma0 = p0 / total
    gamma1 = p1 / total
    return gamma0, gamma1

def M_step(data, gamma0, gamma1):
    '''
    M步：更新模型参数
    '''
    N0 = np.sum(gamma0)
    N1 = np.sum(gamma1)
    mu0 = np.sum(gamma0 * data) / N0
    mu1 = np.sum(gamma1 * data) / N1
    sigma0 = np.sqrt(np.sum(gamma0 * (data - mu0)**2) / N0)
    sigma1 = np.sqrt(np.sum(gamma1 * (data - mu1)**2) / N1)
    alpha0 = N0 / len(data)
    alpha1 = N1 / len(data)
    return alpha0, mu0, sigma0, alpha1, mu1, sigma1

def EM(data, alpha0, mu0, sigma0, alpha1, mu1, sigma1, max_iter=1000, epsilon=1e-6):
    '''
    EM算法主循环
    '''
    for i in range(max_iter):
        gamma0, gamma1 = E_step(data, alpha0, mu0, sigma0, alpha1, mu1, sigma1)
        alpha0_new, mu0_new, sigma0_new, alpha1_new, mu1_new, sigma1_new = M_step(data, gamma0, gamma1)

        # 收敛条件：所有参数变动都小于 epsilon
        if all([
            np.abs(mu0_new - mu0) < epsilon,
            np.abs(mu1_new - mu1) < epsilon,
            np.abs(sigma0_new - sigma0) < epsilon,
            np.abs(sigma1_new - sigma1) < epsilon,
            np.abs(alpha0_new - alpha0) < epsilon,
            np.abs(alpha1_new - alpha1) < epsilon
        ]):
            break

        # 更新参数
        alpha0, mu0, sigma0, alpha1, mu1, sigma1 = alpha0_new, mu0_new, sigma0_new, alpha1_new, mu1_new, sigma1_new

    return alpha0, mu0, sigma0, alpha1, mu1, sigma1

if __name__ == '__main__':
    start = time.time()

    # 初始化参数
    alpha0, mu0, sigma0 = 0.3, -2, 0.5
    alpha1, mu1, sigma1 = 0.7, 0.5, 1

    # 生成数据
    data = generate_data(mu0=0, sigma0=1, mu1=5, sigma1=1, alpha0=0.5, alpha1=0.5)

    # 运行EM算法
    alpha0, mu0, sigma0, alpha1, mu1, sigma1 = EM(data, alpha0, mu0, sigma0, alpha1, mu1, sigma1)

    print("EM算法迭代结果：")
    print(f"alpha0: {alpha0:.4f}, mu0: {mu0:.4f}, sigma0: {sigma0:.4f}")
    print(f"alpha1: {alpha1:.4f}, mu1: {mu1:.4f}, sigma1: {sigma1:.4f}")

    end = time.time()
    print(f"运行时间：{end - start:.4f} 秒")
